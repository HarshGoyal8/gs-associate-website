# GS Associate Website — Sprint 06

> **Sprint:** 06 — Admin Dashboard & CMS
> **Version:** 0.6.0
> **Status:** Planned
> **Estimated Duration:** 10–14 Days

---

# Sprint Goal

Develop a secure Admin Dashboard that allows GS Associate to manage the website without modifying code.

This sprint transforms the project into a self-manageable business platform.

---

# Sprint Objectives

* Build Admin Dashboard
* Configure Authentication
* Implement CMS
* Manage Products
* Manage Brands
* Manage Catalogues
* Manage Website Content
